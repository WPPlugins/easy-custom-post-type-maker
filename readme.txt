= Features =
* Plugin allows you to create Custom Post Types and custom Taxonomies without the need of writing any code.
* Plugin also allows you to manage the Easy Custom Post Type icon.

== Installation ==

1. Upload 'easy-custom-post-type-maker' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the new menu item 'Easy Post Types' to create a Custom Post Type or a custom Taxonomy
